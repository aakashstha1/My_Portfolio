# Portfolio Frontend

This is the **frontend** of my portfolio website built with **React**, **Vite**, **TailwindCSS**, and **Framer Motion**.

## Features
- About myself
- My Projects
- My Achievements with Lottie animations
- My Experience

## Prerequisites
- Node.js >= 18
- npm or yarn

## Installation
```bash
# Clone the repo
git clone <repo-url>

# Navigate to frontend folder
cd client

# Install dependencies
npm install

# Run Locally
npm run dev
